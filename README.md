## 📖 Sobre

O Among Force é um painel desenvolvido para facilitar o gerenciamento da comunidade.

Com ele é possível organizar membros, grupos, aniversários e utilizar o Chat Global em uma interface rápida, moderna e intuitiva.

- 🎂 Organização automática de aniversários

- ⚡ Atualizações constantes
